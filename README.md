Cybersecurity Internship Project

This project was completed as part of a three-week cybersecurity internship.

Week 1

Identified vulnerabilities (Stored XSS, weak password storage, lack of authentication).

Week 2

Implemented password hashing using bcrypt.

Added input validation.

Implemented login and JWT-based authentication.

Added security headers using Helmet.

Week 3

Added logging using Winston.

Performed manual penetration testing.

Created a security best-practices checklist.

Technologies Used

Node.js

Express.js

Bcrypt

JSON Web Token (JWT)

Helmet

Winston
